********

// markup-start

Bold: **two asterisks**

Lists
*****

// markup-start

* Unordered item 1
* Unordered item 2
* Unordered item 3

+ Alternative unordered item 1
+ Alternative unordered item 2
+ Alternative unordered item 3

- Alternative unordered item 1
- Alternative unordered item 2
- Alternative unordered item 3

// markup-end

// markup-start

#. Ordered item 1

2. Alternative Ordered item 2
3. Alternative Ordered item 3
